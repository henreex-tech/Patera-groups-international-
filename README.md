# Patera Group International

This repository contains the landing page for **Patera Group International**, a vertically integrated conglomerate based in Nigeria with core sectors in:

- Real Estate
- Agro Business
- Technology
- Sports

## About
The landing page showcases Patera Group International’s vision, initiatives, and sectors.  
It is fully responsive and designed to work on desktop, tablet, and mobile devices.  

## Features
- Hero section with tagline and call-to-action buttons
- Sectors section with cards and descriptions
- Vision and initiative sections
- Stats and progress tracking
- Custom cursor (desktop only)
- Responsive design for mobile and tablet

1. Clone or download this repository
2. Open `index.html` in a web browser
